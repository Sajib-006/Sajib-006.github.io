Capturing long range nucleotide interaction in sequence-to-methylation modeling

Working with Dr. M Sohel Rahman(BUET) and Dr. Md. Abul Hassan Samee(Baylor College of Medicine)

We propose a hybrid model consisting of convolutional netwrok and recurrent neural network which can predict methylation status better than existing models on genome wide methylation data. We used tomtom to identify the sequence motifs. To visualize sequence region importance, we used deepSHAP scores. We also show that the model can predict accurately for sequence with less nucleotides than our baseline method. The work is still in progress.

Ghost burster using HMM filtering

Technology: Java

Description: A Ghost bursting game using hidden markov model filetring and particle filtering.

Features:

  • Designed a system that updates the probabilities of the ghost being in a grid cell as the time elapses and a sensor measurement observed. To advance a time to the next time stamp, you will press a button and see the change of probabilities in the grid. Then when you press on a cell to get the sensor measurement it will display the color of the grid cell based on the distance to the ghost, and update the probability distribution of all grid cells accordingly. Finally when you are almost sure about the ghost location (the cell with the highest probability, you can hit the cell to catch the ghost.
  • Use the HMM filtering technique to do the task.

Constraint Satisfaction Problems (CSP)

Technology: Java

Description: An order m quasigroup is a Latin square of size m. That is, a m by m multiplication table in which each element occurs once in every row and column. Quasigroup Completion Problem asks to complete a quasigroup given some of its entries. For example, partially specified quasigroup.

Features:

Solved CSP using:

  • Simple Backtracking (BT)
  • Forward Checking (FC)
  • Maintaining Arc Consistency (MAC)
  • Used variable ordering heuristics: Smallest Domain First, max-dynamic-degree, brelaz, domdeg, domddeg, random
